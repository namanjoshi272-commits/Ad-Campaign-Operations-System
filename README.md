# Advertising Campaign Operations & Order Management System

## Project Overview
This portfolio project simulates an end-to-end operational workflow within a fast-paced Advertising & Marketing agency. Utilizing a messy, 100-row transactional dataset capturing regional and international ad space orders, I engineered a standardized database structure to eliminate data latency, calculate SLA turnarounds, and generate executive MIS reports.

## Key Features Built
* **Data Sanitization & Cleaning:** Standardized mixed textual text casing using `PROPER` string logic and repaired faulty regional chronological data strings using Excel's text-parsing utilities.
* **Relational Database Model:** Constructed automated client and vendor communication paths using advanced `XLOOKUP` arrays, completely mitigating downstream delivery anomalies.
* **Executive MIS Dashboard:** Developed live Pivot Tables and visual charts mapping budget spending models (INR Lakhs) and order status tracking.
* **Process Documentation:** Developed a formal Standard Operating Procedure (SOP-ADV-OPS-001) incorporating a high-priority risk mitigation escalation matrix.

## Repository Contents
* `Advertising_Operations_System_2026.xlsx`: The core analytical workbook containing the master dataset, lookup logic, and executive metrics dashboard.
* `SOP-ADV-OPS-001.pdf`: Standard Operating Procedure manual mapping the exact intake and validation framework.
